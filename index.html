AT&T Integrated Cloud + Network Virtualization – AIC Gamma

AIC is the effort to build a common hardware and software platform with a goal to reduce the overall cost for AT&T and be able to support different Cloud services. AIC Gamma adds the Network Virtualization Platform with support for virtualized Customer Edge (vCE) and virtualized Provider Edge (vPE) routers on commodity x86 hardware. CE and PE routers have traditionally been implemented on physical router platforms from vendors such as Juniper and Cisco. These routers apply various controls to traffic as it enters the AT&T network before passing the traffic to the core of the network. Packet forwarding and data plane throughput are major concerns, but NVP and now AIC pursue the shift towards decoupling control plane from data plane and shifting functions from dedicated hardware into software deployed elastically onto a general purpose (typically x86) computing platform. AT&T Cloud provides third party developers with servers, storage, networking, industry standard APIs, and the tools to build fault-tolerant cloud applications with low latency and high scalability. The platform allows software developers to build applications quickly and bring them online without worrying about capacity planning, scaling, security, or performance.
